1枚の紙が、蛇腹の単語帳になるまで

印刷される A4 1ページと、切り離して折ったあとの完成イメージを図にしました。 赤線が「切る」、青い破線が「折る」です。帯の本数・パネルの高さは実際にはアプリの設定で自由に変えられます（この図は4本の例）。

1 印刷される1ページ（A4）

黒一色・最小限のインクで印刷します。用紙マージンは0（フチなし）で、帯は紙の端いっぱいまで使います。 縦の切り取り線で4本の帯に切り分け、各帯は上から順に「表面 → 裏面 → 表面 → 裏面…」と並びます。ペアの途中で帯や紙が切り替わることはありません。 罫線は紙全面には引かず、端の小さな目印（トンボ）だけにしてインクを節約します。定規を目印同士に合わせて折る／切ります。

  • 切り取りの目印（上下の端）
  • 折りの目印（帯ごとの左右の端）
帯の数：4（可変） パネル高さ：可変 用紙マージン0（フチなし） 黒一色・端の目印のみ ページ番号表示あり

2 帯を切り離して蛇腹に折ると…

1本の帯を折り線どおりに山折り・谷折りと交互にジグザグへ折りたたむと、アコーディオン状に立つ単語帳になります。 手前を向く面（表面）と奥を向く面（裏面）が交互に現れ、パタパタと1コマずつめくって答え合わせができます。

  • 手前を向く面（実線）＝単語
  • 奥を向く面（破線）＝訳

3 Web版の入力欄イメージ

入力欄は「表面」「裏面」の2列テーブル。スプレッドシートで2列選択してコピーし、そのまま貼り付けられます。 現在のレイアウト設定（帯の数・パネル高さ）から自動計算し、ページの境目には太い区切り線を引きます（行は使いません）。

表面裏面
  • 太い区切り線 ＝ ここまでが1ページ目

スプレッドシートで2列を選択 → Cmd/Ctrl+C → 表の左上セルをクリックして Cmd/Ctrl+V するだけで一括入力できます。

単語数が帯の残りパネルに収まらない場合は、そのペアごと次の帯・次のページへ送られます。印刷ページの右下に「P. 1 / 3」のようなページ番号が入るので、どこで紙が分かれるか一目で分かります。 Web版の入力テーブルでも同じ区切りが「ここまでが1ページ目」として表示されるので、印刷結果とズレなく確認できます。
